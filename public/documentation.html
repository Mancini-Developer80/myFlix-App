My Flix API documentation

Welcome!

Welcome to the documentation for myFlix API. Here you will find all the information you need to interact with the API and access the data stored in the database.

API endpoints Methods Request Response
/movies GET none Returns a list of all movies to the user.
/movies/title/:title GET title Returns data (description, genre, director, image URL, whether itâ€™s featured or not) about a single movie by title to the user.
/genres/:name GET name Returns data (description) about a genre by name/title.
/directors/:name GET name Returns data (bio, birth year, death year) about a director by name.
/users/register POST username, password, email Allows new users to register.
/users/update PUT username Allows users to update their user info (username).
/users/favorites/:title POST title Allows users to add a movie to their list of favorites (showing only a text that a movie has been added).
/users/favorites/:title DELETE title Allows users to remove a movie from their list of favorites (showing only a text that a movie has been removed).
/users/deregister DELETE email Allows existing users to deregister (showing only a text that a user email has been removed).
