My Flix API documentation

Welcome!

Welcome to the documentation for myFlix API. Here you will find all the information you need to interact with the API and access the data stored in the database.

API endpoints Methods Request Response
/users GET None Returns a list of all users
/users/:id GET None Returns data about a single user by ID
/users POST JSON: { "username": "string", "email": "string", "password": "string" } Registers a new user
/users/:id PUT JSON: { "username": "string", "email": "string", "password": "string" } Updates user info by ID
/users/:id/movies/:movieTitle POST None Adds a movie to a user's list of favorites
/users/:id/movies/:movieTitle DELETE None Removes a movie from a user's list of favorites
/users/:id DELETE None Deregisters an existing user
/movies GET None Returns a list of all movies
/movies/:title GET None Returns data about a single movie by title
/genres/:name GET None Returns data about a genre by name
/directors/:name GET None Returns data about a director by name
/directors POST JSON: { "Name": "string", "Bio": "string", "Birth": "date", "Death": "date" } Adds a new director
/movies POST JSON: { "Title": "string", "Description": "string", "Genre": { "Name": "string", "Description": "string" }, "Director": { "Name": "string", "Bio": "string", "Birth": "date", "Death": "date" }, "ImageURL": "string", "Featured": "boolean" } Adds a new movie
